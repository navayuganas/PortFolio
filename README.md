## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required!

## Customization

### Personal Information
- Update your name, role, and description in the Hero section
- Modify the About section with your own information
- Update statistics in the About section

### Skills
- Add or remove skills in the Skills section
- Update icons using Font Awesome classes

### Projects
- Replace project cards with your own projects
- Update project descriptions, tags, and links
- Modify project placeholders with actual images

### Experience
- Update the timeline with your work experience
- Add or remove timeline items as needed

### Contact
- Update contact information (email, phone, location)
- Modify social media links
- The contact form currently shows a success message (integrate with your backend)

### Colors
- Customize colors by modifying CSS variables in `styles.css`:
  ```css
  :root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... */
  }
  ```
